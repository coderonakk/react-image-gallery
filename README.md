# React Image Gallery

A simple image gallery built with React and Axios that fetches images from the Picsum API and displays them with client-side pagination.

## Features

* Fetch images from an external API
* Pagination with Next and Previous controls
* Dynamic rendering using React state
* Loading state while fetching data
* Responsive image gallery layout
* Built with React Hooks

## Tech Stack

* React
* JavaScript
* Axios
* Tailwind CSS
* Vite

## What I Learned

* useState
* useEffect
* API fetching with Axios
* Conditional rendering
* Array mapping with React
* Pagination logic
* State-driven UI updates
